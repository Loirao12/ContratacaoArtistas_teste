# 🎤 Artist Hiring Platform

Este projeto foi desenvolvido como parte do **EnterScience Internship Technical Test – 2025**.  
A aplicação permite que usuários pesquisem artistas, criem contratos para eventos privados e gerenciem contratos já submetidos.

---

## 🧩 Project Overview

A plataforma simula um sistema onde o usuário pode:

- 🔍 Pesquisar artistas disponíveis  
- 📝 Selecionar um artista e preencher um formulário de contrato  
- 📤 Submeter o contrato  
- 📂 Visualizar contratos criados anteriormente  

A aplicação é dividida em **frontend** e **backend**, seguindo boas práticas de separação de responsabilidades.

---

## 🛠️ Technologies Used

### Frontend
- ⚡ React (Vite)  
- 🟦 TypeScript  
- 🔗 React Router DOM  
- 🎨 CSS (custom styling, responsive layout)  

### Backend
- 🌐 Laravel  
- 🐘 PostgreSQL  
- 🔌 REST API
